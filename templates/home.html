Korean abstractive summary service
Building a generative summary model using natural language processing technology
This project provides a text summary function using natural language processing technology. When the user passes the input text to the system, the text summary result is returned using the learned generative summary model.
Text summarization
Abstractive
Input
연구 결과에 따르면 일상적인 운동은 우리의 신체적인 건강을 유지하고 개선하는 데 매우 중요한 역할을 합니다. 또한, 정신적인 면에서도 운동은 스트레스 감소와 기분 개선에 도움을 줄 수 있습니다.
Output
· 일상적인 운동은 신체적, 정신적 건강에 긍정적인 영향을 줍니다.
생성요약이란, 대상 문서의 의미를 해석하여 지능적으로 요약문을 만드는 방법으로, 완전히 새로운 문장을 생성한다.
Extractive
Input
연구 결과에 따르면 일상적인 운동은 우리의 신체적인 건강을 유지하고 개선하는 데 매우 중요한 역할을 합니다. 또한, 정신적인 면에서도 운동은 스트레스 감소와 기분 개선에 도움을 줄 수 있습니다.
Output
· 운동은 신체와 정신 건강에 도움이 됩니다.
추출요약이란, 대상 문서에서 중요한 문장을 식별하여 요약문을 만드는 방법으로, 원본에서의 정확한 문장을 포함하는 요약이다.
Procedure
Data collection and preprocessing
· 다양한 주제와 장르의 문서 및 문장 데이터를 수집하고 전처리하여 정제된 학습 데이터를 구성

·SentencePiece tokenizer를 사용하여 토큰화를 수행
model design
· 인코더-디코더 구조를 활용한 요약 모델을 구현

· T5 모델을 사용하여 인코더에서 문맥 정보를 추출하고 디코더에서 요약 문장을 생성

· 어텐션 메커니즘을 사용하여 인코더의 출력에 집중
test
· 개발된 요약 모델에 대해 테스트를 수행

· ROUGE SCORE를 사용하여 요약 결과를 평가
learning and assessment
· 수집한 학습 데이터를 사용하여 모델을 학습

· 전이학습을 이용하여 사전에 학습된 모델을 활용

·ROUGE SCORE를 사용하여 모델의 성능을 평가
wep page
· 웹페이지를 구현하여 사용자가 텍스트를 입력하고 요약 결과를 확인
